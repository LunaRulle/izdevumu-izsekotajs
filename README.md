# Izdevumu izsekotājs

Izdevumu sekotājs ir vienkārša programma rakstīta python, kas atļauj tev ievadīt un saglabāt savus izdevumus. Programmā ir iespējams ievadīt jaunus izdevumus un dzēst tos, apskatīt visus izdevumus, apskatīt kopsavilkumus par kategorijām, kā arī aspkatīt izdevumus pa mēnešiem.
Ja vēlaties jūs varat arī eksportēt izdevumus csv failā.

## Uzstādīšana

Lai uzstādītu programmu jums vajadzēs Python 3.7+ un Git
Jūs var vienkārši palaist šo kommandu lai sāktu strādāt
`git clone https://github.com/LunaRulle/izdevumu-izsekotajs && cd izdevumu-izsekotajs/expense_tracker && python app.py`

## Lietošana

Jums būs pieejamas šīs opcijas lietojot programmu
1) Pievienot Izdevumu
2) Parādīt izdevumus
3) Filtrēt pēc mēneša
4) Kopsavilkums pa kategorijām
5) Dzēst izdevumu
6) Eksportēt uz csv
7) Iziet

Vienkārši termināli ierakstiet ciparu kas atiecas uz iespēju kuru vēlaties izvēlēties un spiediet enter lai palaistu to. Jums būs pastāstīts vai parādīts kāda veida formātā jus veikt iespējas.
